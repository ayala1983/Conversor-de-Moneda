=======================================
 CONVERSOR DE MONEDAS EN JAVA - README
=======================================

Descripción:
------------
Este proyecto es un conversor de monedas que utiliza la API de ExchangeRate (https://www.exchangerate-api.com)
para obtener el tipo de cambio en tiempo real. Permite convertir entre monedas y guardar un historial 
de conversiones realizadas.

Estructura del Proyecto:
------------------------
- Main.java
- ConsultaTipoDeMoneda.java
- TipoDeCambio.java
- TipoDeMonedaNoEncontradaException.java
- json.jar (librería externa en /lib)
- errores.log (errores de conexión)
- historial.txt (historial de conversiones)

Requisitos:
-----------
- JDK 8 o superior
- IDE como IntelliJ
- Librería json.jar
- Conexión a internet

Instrucciones de Uso:
---------------------
1. Ejecutar el programa.
2. Elegir una opción del menú.
3. Ingresar el monto.
4. Ver resultado en consola.
5. Ver historial con la opción 8.
